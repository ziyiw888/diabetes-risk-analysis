# Diabetes Risk Analysis

## Overview
This project analyzes social and behavioral factors associated with diabetes risk using real-world datasets.

## Goals
- Explore relationships between income, BMI, and diabetes prevalence
- Perform exploratory data analysis (EDA)
- Identify key predictors of diabetes

## Methods
- Data cleaning and preprocessing using Pandas
- Visualization using Matplotlib
- Statistical analysis of demographic factors

## Key Findings
- Higher BMI is strongly associated with increased diabetes risk
- Income level shows correlation with health outcomes
- Preventive healthcare access varies across groups

## Files
- `diabete.py`: Main analysis script
- `test.py`: Test cases
- `eda.pdf`: Exploratory data analysis report
- `*.png`: Visualization outputs
